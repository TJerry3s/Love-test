# 恋爱倾向测试

一款支持手机端与PC端网页适配的趣味恋爱风格测试工具，帮助用户探索自己的恋爱倾向和行为模式。

![Version](https://img.shields.io/badge/version-1.0.0-pink)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 产品特点

- 🎯 **20道精心设计的题目**：基于爱情三元论和恋爱心理学研究
- 📊 **5个测试维度**：激情型、理智型、奉献型、依赖型、独立型
- 💕 **9种恋爱类型**：浪漫激情型、理性务实型、无私奉献型、甜蜜依恋型、独立平等型、完美平衡型、理智浪漫型、温和陪伴型、谨慎保守型
- 📈 **五维雷达图可视化**：直观展示各维度得分分布
- 💡 **个性化恋爱建议**：择偶建议、相处建议、自我成长建议
- 📱 **双端适配**：完美支持手机端和PC端浏览
- 🔒 **隐私保护**：数据仅存储在本地浏览器，不上传任何个人信息

## 🚀 快速开始

### 在线使用

直接打开 `index.html` 文件即可开始测试。

### 本地运行

1. 克隆或下载本项目
2. 使用浏览器打开 `index.html` 文件
3. 点击"开始测试"按钮

**注意**：由于使用了 ES6 模块导入功能，建议通过本地服务器运行。

#### 使用 Python 启动本地服务器

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中访问 `http://localhost:8000`

#### 使用 Node.js 启动本地服务器

```bash
# 安装 http-server
npm install -g http-server

# 启动服务器
http-server -p 8000
```

## 📊 测试维度

| 维度 | 说明 |
|------|------|
| 🔥 激情型 | 反映在恋爱中对心动体验、浪漫感受和外在吸引的重视程度 |
| 🧠 理智型 | 反映在恋爱中对现实条件、未来规划和理性决策的重视程度 |
| 🎁 奉献型 | 反映在恋爱中愿意为对方付出的程度和自我牺牲的倾向 |
| 🤝 依赖型 | 反映在恋爱中对情感陪伴和依赖的需求程度 |
| 🌿 独立型 | 反映在恋爱中保持自我空间和独立性的需求程度 |

## 💕 恋爱类型

根据5个维度的得分组合，系统会判断用户属于以下9种恋爱类型之一：

1. **浪漫激情型** 💖 - 追求心动体验，相信一见钟情
2. **理性务实型** 🧠 - 理性规划，注重现实基础
3. **无私奉献型** 🎁 - 全心付出，无私奉献
4. **甜蜜依恋型** 🤝 - 渴望陪伴，情感依赖
5. **独立平等型** 🌿 - 保持自我，平等相处
6. **完美平衡型** ⚖️ - 全面发展，平衡和谐
7. **理智浪漫型** 💫 - 理性与浪漫并存
8. **温和陪伴型** 🌸 - 温和包容，温暖陪伴
9. **谨慎保守型** 🔒 - 谨慎观察，慢慢投入

## 📝 开发说明

### 修改题目

编辑 `data/questions.js` 文件中的 `questions` 数组。

### 修改恋爱类型数据

编辑 `utils/calculator.js` 文件中的 `loveTypeData` 对象。

### 修改样式

所有样式都在 `index.html` 的 `<style>` 标签中，可以根据需要调整。

### 添加新的恋爱类型

1. 在 `utils/calculator.js` 的 `loveTypeData` 中添加新类型数据
2. 在 `determineLoveType` 函数中添加判断逻辑
